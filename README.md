# Covid-Data-Analysis-Project
A detailed Python-based exploratory data analysis (EDA) of COVID-19 data trends in India. This project plots state-wise infection rates, active cases, and vaccination progress using Pandas, Seaborn, and Plotly to highlight major pandemic hotspots and trends.


Tech Stack
Programming Language: Python
Libraries: Pandas (Data Manipulation), NumPy (Numerical Computing).
Visualization: Matplotlib and Seaborn (Statistical Data Visualization).
Environment: Jupyter Notebook.


Data Source
The project utilizes a formatted COVID-19 dataset with daily reports of confirmed cases, deaths, and recovered cases indexed by country and date.


Features & Highlights
Data Cleaning: Missing data handling and date-time object formatting for time-series analysis.
Aggregated Metrics: Calculation of total global cases, active cases, and mortality rates.
Trend Visualization: Time-series line plots illustrating the growth of the pandemic over a certain period.
Comparative Analysis: Bar charts and heatmaps illustrating the effects of the pandemic in the most affected countries.
